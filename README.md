# Spotify Clone Project

This project is a Spotify clone built with HTML, CSS, and JavaScript, aimed at providing users with a familiar and enjoyable music streaming experience.

## Features
- Play songs from different albums
- Responsive design with a hamburger menu for mobile navigation
- Volume control, seekbar, and play/pause functionality
- Previous and next buttons for easy navigation between tracks

## Technologies Used
- HTML5 for markup
- CSS3 for styling and layout
- JavaScript for interactive functionality
